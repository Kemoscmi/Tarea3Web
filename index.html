Diseño Web Responsivo

Permite que un sitio se adapte automáticamente a diferentes tamaños de pantalla. Es clave para ofrecer una experiencia fluida y accesible en cualquier dispositivo.

El diseño web responsivo se basa en tres principios fundamentales:

  • Rejillas fluidas: Usar porcentajes en lugar de píxeles para que el ancho del contenido se adapte.
  • Imágenes flexibles: Las imágenes se redimensionan automáticamente según el dispositivo.
  • Media Queries: Permiten aplicar estilos dependiendo del ancho de pantalla.

Viewport

Es el área visible de la página en el navegador. Se controla con esta metaetiqueta:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Sin definir el viewport, los navegadores móviles muestran el sitio como una miniatura del diseño de escritorio, forzando al usuario a hacer zoom. Esta etiqueta garantiza que el diseño se ajuste correctamente.

Media Queries

Permiten aplicar estilos según las características del dispositivo, como su ancho:

@media (max-width: 768px) {
  body {
    background-color: #f4f4f4;
  }
}

Las media queries permiten modificar la apariencia del sitio según el tamaño de la pantalla. Con ellas puedes:

  • Ocultar secciones en móvil.
  • Reordenar bloques.
  • Aplicar estilos distintos según el dispositivo.

Bootstrap 5

  • Container: centra y da margen al contenido.
  • Grid System: divide el diseño en columnas responsivas.
  • Flex: organiza elementos fácilmente.
  • Carousel: muestra imágenes de forma dinámica.
  • Navbar: barra de navegación adaptable.

Bootstrap 5 facilita la creación de sitios responsivos con componentes listos para usar. Su sistema de 12 columnas, utilidades de flexbox y elementos interactivos lo hacen ideal para cualquier tipo de sitio web moderno.
